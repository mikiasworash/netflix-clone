## Getting Started

### 1. Clone the repository

```
git clone https://github.com/mikiasworash/netflix-clone.git
```

### 2. Install Dependencies

```
cd netflix-clone

npm install
```

### 3. Run the development server

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
